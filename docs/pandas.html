9.1 創建pd.DataFrame

9.1.1 將dictionary轉成data.frame

9.1.1.1 法一： 像R那樣做

  • 回憶一下，在R裡面，怎麼自己建立data.frame? 通常都是這樣做： data = data.frame(a = c(), b = c(), c = c())
  • 當我們這樣做的時候，其實對應到dictionary，不就是一個一個的key value嗎？ 所以概念完全一樣：
import pandas as pd
data_dic = {
     'Country': ['Belgium', 'India', 'Brazil'],
     'Capital': ['Brussels', 'New Delhi', 'Brasília'],
     'Population': [11190846, 1303171035, 207847528]
}
data_df = pd.DataFrame(data_dic)
data_df
#>    Country    Capital  Population
#> 0  Belgium   Brussels    11190846
#> 1    India  New Delhi  1303171035
#> 2   Brazil   Brasília   207847528

9.1.1.2 法二： 對付API request下來的那種JSON

9.1.2 將2d-array轉成data.fream

  • 在R裡面，我們有時也會先有一個matrix，然後把他轉成data.frame，做法是：as.data.frame(matrix)
  • 那既然pandas是建立在numpy上，那也可將2-dim array轉成dataframe，做法也雷同： pd.DataFrame(matrix)
import numpy as np
matrix = np.array([[1,2],[3,4]])
print(matrix)
#> [[1 2]
#>  [3 4]]
data_pd2 = pd.DataFrame(matrix)
print(data_pd2)
#>    0  1
#> 0  1  2
#> 1  3  4
  • 不過，把matrix轉成data.frame後，我們還缺少column name，所以，在R裡，都還會做colnames(data) = c()
  • 那在pd裡，我們該如何把現有的dataframe加上欄位名稱呢？這個後面再講

9.2 Select

9.2.1 [] & [[]]

  • 如果今天用[]來select出一個columm，那做出來的type會是series，不是data.frame
one_bracket = cars["country"]
print(one_bracket)
#> US     United States
#> AUS        Australia
#> JAP            Japan
#> IN             India
#> RU            Russia
#> MOR          Morocco
#> EG             Egypt
#> Name: country, dtype: object
print(type(one_bracket))
#> <class 'pandas.core.series.Series'>
  • 但如果用兩個brackets [[]]，就會select出一個data.frame
two_bracket = cars[["country"]]
print(two_bracket)
#>            country
#> US   United States
#> AUS      Australia
#> JAP          Japan
#> IN           India
#> RU          Russia
#> MOR        Morocco
#> EG           Egypt
print(type(two_bracket))
#> <class 'pandas.core.frame.DataFrame'>
  • 而且，兩個bracket，恰好用在你要select多個column
two_bracket2 = cars[["country", "drives_right"]]
print(two_bracket2)
#>            country  drives_right
#> US   United States          True
#> AUS      Australia         False
#> JAP          Japan         False
#> IN           India         False
#> RU          Russia          True
#> MOR        Morocco          True
#> EG           Egypt          True
  • 最後，如果要切row出來，那因為row代表的是index，所以會用數字切 ，例如，切最開始到第三個index:
print(cars[:3])
#>      cars_per_cap        country  drives_right
#> US            809  United States          True
#> AUS           731      Australia         False
#> JAP           588          Japan         False

9.2.2 loc and iloc

  • loc and iloc是pandas的method，所以是用data.loc[列名,行名], data.iloc[列index,行index]來運作
  • loc是吃key的(i.e. column name, row name)，iloc是吃index(i.e. number)
cars
#>      cars_per_cap        country  drives_right
#> US            809  United States          True
#> AUS           731      Australia         False
#> JAP           588          Japan         False
#> IN             18          India         False
#> RU            200         Russia          True
#> MOR            70        Morocco          True
#> EG             45          Egypt          True
  • 抓出列名是JAP, 行名是drives_right的資料
print(cars.loc["JAP","drives_right"])
#> False
print(cars.iloc[2,2])
#> False
  • 抓出列名是RU, MOR，行名是drives_right的資料
print(cars.loc[["RU", "MOR"],"drives_right"])
#> RU     True
#> MOR    True
#> Name: drives_right, dtype: bool
print(cars.iloc[[4,5],2])
#> RU     True
#> MOR    True
#> Name: drives_right, dtype: bool
  • 以series的type，抓出drives_right這個column
print(cars.loc[:,"drives_right"])
#> US      True
#> AUS    False
#> JAP    False
#> IN     False
#> RU      True
#> MOR     True
#> EG      True
#> Name: drives_right, dtype: bool
print(cars.iloc[:,2])
#> US      True
#> AUS    False
#> JAP    False
#> IN     False
#> RU      True
#> MOR     True
#> EG      True
#> Name: drives_right, dtype: bool
  • 以DataFrame的type，抓出drives_right這個column
print(cars.loc[:, ["drives_right"]])
#>      drives_right
#> US           True
#> AUS         False
#> JAP         False
#> IN          False
#> RU           True
#> MOR          True
#> EG           True
print(cars.iloc[:,[2]])
#>      drives_right
#> US           True
#> AUS         False
#> JAP         False
#> IN          False
#> RU           True
#> MOR          True
#> EG           True
